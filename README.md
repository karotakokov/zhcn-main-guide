# zhcn-main-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面本身不针对任何单一域名或具体网站，而是作为一组可独立访问的静态资源进行维护。

## 项目简介

这是一个以文档和页面归档为主要用途的仓库。其中的 HTML 文件通常包含说明性内容、工具界面或信息展示页，方便通过浏览器直接打开查看。仓库本身不提供后端服务或动态生成功能，所有页面均为纯静态 HTML。

## 目录说明

仓库根目录下以文件夹或文件形式组织页面，每个页面通常包含以下内容：

- 一个独立的 HTML 文件，可直接在浏览器中运行
- 部分页面可能附带 CSS 或 JavaScript 资源（内联或同目录下）
- 子目录可能用于存放相关的辅助文件

目录结构示例：

```
/
├── page-example.html
├── another-page/
│   ├── index.html
│   └── style.css
└── README.md
```

具体页面数量与结构可能随时间更新，请以仓库实际文件为准。

## 页面归档说明

归档的 HTML 页面具有以下特征：

- 不依赖外部网络资源（除非特别说明）
- 可在本地直接打开使用
- 内容保持稳定，不频繁变动
- 不包含任何形式的跟踪、统计或第三方调用

每个页面的用途和说明通常写在文件开头的注释中，或通过页面本身的内容体现。

## 维护说明

本仓库由维护者不定期更新。更新内容可能包括：

- 新增页面
- 对已有页面的小幅修正
- 清理不再需要的文件

欢迎通过 Issues 提出建议或报告问题，但请注意：

- 本仓库不提供与具体域名或网站相关的技术支持
- 不接受商业推广或营销性质的提交
- 提交内容请保持简洁、直接

## 使用方式

1. 克隆或下载本仓库到本地
2. 用浏览器打开任意 HTML 文件即可查看
3. 也可通过 GitHub Pages 等静态托管服务部署后在线访问
